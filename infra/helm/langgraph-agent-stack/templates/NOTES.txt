LangGraph Agent Stack — Post-Install Notes

WARNING: Default config uses SQLite (replicaCount=1).
    SQLite only supports a single writer.

To scale horizontally, use Redis:
  helm upgrade {{ .Release.Name }} . \
    --set config.memoryBackend=redis \
    --set secrets.redisUrl=redis://your-redis:6379/0 \
    --set replicaCount=3

Access the API:
  kubectl port-forward svc/{{ include "langgraph-agent-stack.fullname" . }} 8080:80 -n {{ .Values.namespace.name }}
  curl http://localhost:8080/health
